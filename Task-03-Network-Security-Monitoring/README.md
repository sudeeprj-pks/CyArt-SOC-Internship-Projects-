# Task 03 – Network Security Monitoring and Analysis

## Tools Used
- Wireshark
- Snort IDS
- Scapy
- Kali Linux

## Objectives
Analyze network traffic, configure intrusion detection rules, and generate test packets.

## Wireshark Analysis
Captured traffic and identified TCP, UDP, and ICMP protocols.

## Snort IDS
Created rules to detect SYN packets and port scanning activity.

## Scapy Script
Developed Python script to generate SYN packets to simulate scanning behavior.

## Outcome
Snort successfully detected simulated attack traffic.
