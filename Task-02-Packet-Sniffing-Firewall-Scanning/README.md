# Task 02 – Packet Sniffing, Firewall Configuration and Vulnerability Scanning

## Objective
This task demonstrates network monitoring, firewall configuration, and vulnerability scanning using Kali Linux tools.

## Tools Used
- Python (Scapy)
- Matplotlib
- iptables
- OpenVAS (Greenbone Vulnerability Manager)
- Kali Linux

## Project Components

### 1. Packet Sniffer
A Python script using Scapy to capture network packets and identify protocols.

### 2. Protocol Analysis
Captured packets were analyzed and visualized using a protocol distribution chart.

### 3. Firewall Configuration
iptables rules were configured to allow only specific ports (SSH and HTTP).

### 4. Vulnerability Scanning
OpenVAS was installed and configured to perform vulnerability assessments.
