# Network Security Monitoring & SIEM Lab (Task 05)

## Project Overview
This project demonstrates practical Security Operations Center (SOC) tasks including network monitoring, traffic analysis, troubleshooting, SIEM setup, and threat detection using ELK Stack.

The objective of this lab is to simulate a real-world security monitoring environment and detect suspicious activities such as SSH brute-force attacks.

---

## Tools Used

- Ubuntu Linux
- Cisco Packet Tracer
- Wireshark
- tcpdump
- ELK Stack (Elasticsearch, Logstash, Kibana)
- Hydra (Brute Force Tool)
